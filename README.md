# Termux-Locker
This is a tool to protect termux.

# Intsallation
1) `pkg update`

2) `pkg upgrade`

3) `pkg install git`

4) `git clone https://github.com/krishtechstudio/Termux-Locker/`

5) `cd Termux-Locker`

6) `chmod +x install.sh`

7) `./install.sh`
